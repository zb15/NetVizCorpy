This folder contains the following
B2BIdentifyQUIDs: class and functions to search company qids given a company name
B2BQuery: class and functions to do web scraping
B2BDeriveNetwork: class and functions to build company networks given qids and search levels
B2BPostProcess: class and functions to clean the wiki data
B2BVisualiseNetwork: class and functions to prepare and perform visualisation
main_script: example usage
